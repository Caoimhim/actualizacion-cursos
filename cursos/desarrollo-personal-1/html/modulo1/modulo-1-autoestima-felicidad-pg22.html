Tema 4. Emprendiendo mi superación personal

1. Define una meta clara y concreta

Cómo mencionaba el ejemplo anterior, debes definir qué es lo que quieres hacer para cumplir una meta.
Debes de tomar en cuenta lo siguiente:


  • Sincera: Define con exactitud qué es lo que quieres.

  • Personal: Debe surgir de tu interior, no impuesta por alguien externo.

  • Realista: Que la puedas alcanzar en un corto pazo.

  • Divisible: Determina pasos concretos para alcanzarla, como pequeñas metas.

  • Medible: Que tenga una medida (día, hora, etc.) Por ejemplo: este mes le dedicaré media hora del día a aprender inglés.

2. Establece las tareas que debes realizar para lograrlas

Con el ejemplo del libro, estaríamos en el paso de pensar qué debemos hacer para conseguirlo.

Ya establecida la meta, debes tomar en cuenta los pasos necesarios para alcanzarla. Tienes que planear y realizar paso a paso de manera constante, no de un momento a otro o de manera improvisada. Asegúrate de tener estos esfuerzos constantes para que esa meta o proyecto se esté desarrollando y realmente haya un progreso por cumplirlo.
