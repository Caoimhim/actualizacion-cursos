Ejercicio: Insertar Fórmulas

Objetivo: Repasar cómo insertar fórmulas en un Excel.
Instrucciones:

  1. Descargar el siguiente documento: Ejercicio Excel
  2. Siga los pasos para realizar el ejercicio.
  3. En la celda F5 se pondrá el signo de "=".
  4. Sin presionar ninguna tecla, seleccione la celda D5 y agregue el signo de "-" (resta).
  5. A continuación seleccione la celda E5.
  6. Presione la tecla Enter.
  7. De manera inmediata la fórmula se aplica y se muestra el resultado.
