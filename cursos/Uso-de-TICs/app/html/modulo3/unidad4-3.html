Comenzar a Trabajar con Excel

Para empezar a trabajar con Excel es necesario identificar qué se ve en una hoja de Excel.

  • Cuando abre Excel, abre un libro de trabajo, y este libro puede tener varias hojas, así como se ve en la parte inferior del gráfico.
  • Una hoja de Excel tiene columnas y filas.
  • En Excel se pueden introducir datos, como son: números y textos.
  • Con esos datos se pueden tener fórmulas para realizar algunos cálculos, y tener gráficas para tener la forma visual de esa información.
