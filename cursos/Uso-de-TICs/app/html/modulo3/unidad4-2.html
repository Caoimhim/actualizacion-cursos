Hoja de Cálculo

Se le llama hoja de cálculo a un programa que permite manejar números y letras distribuidos en tablas. Con estos datos usted puede realizar cálculos, es decir, utilizar algunas fórmulas y funciones para obtener otros resultados, e incluso utilizar algún tipo de gráficos con esos datos para tener esta información de una forma más visual.

A la hoja de cálculo Microsoft se le llama "Excel".

En esta unidad usted aprenderá cómo usar la hoja de cálculo de Microsoft Office, aprovechando al máximo cada una de las funciones con las que cuenta esta aplicación.

Tenga la certeza de que todo lo que verá en esta unidad le será de gran utilidad en sus actividades diarias, tanto personales como profesionales.
