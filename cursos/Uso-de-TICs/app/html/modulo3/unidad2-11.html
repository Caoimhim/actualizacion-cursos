Revisar Ortografía y Gramática

La ortografía y la gramática son muy importantes al momento de elaborar un documento.

Una de las funciones de Word es detectar cuando una palabra está mal escrita o tiene una falta de ortografía. Cuando sucede esto, Word lo hace notar con un subrayado en rojo.

Ejemplo:

Para que Word haga las correcciones, es necesario indicarle primero el idioma en el que se trabajará el documento, los pasos a seguir para eso son:

  1. Seleccione del menú "Revisar" (Review) la opción de "Definir idioma" (Set Proofing Language ) del apartado de Lenguaje (Language).



    Aparecerá una ventana con las distintas alternativas a seleccionar: 3. De la lista que aparece, seleccione el idioma que desee. En el caso de México se utiliza el Español (México), Spanish Mexico.

  2. Presione "Aceptar" (OK) al finalizar.

Ahora si, una vez que se define el lenguaje, se tienen dos opciones para verificar la ortografía y gramática del documento:

  • Revisión Automática
  • Revisión Manual

Cuando existen errores de ortografía o gramática en el texto, Microsoft Word los marca con un color para que el usuario pueda verificar los errores. Los errores de ortografía se marcan con rojo y los de gramática con azul.

Microsoft Word ofrece sugerencias para corregir estos errores.

A continuación se muestra el procedimiento a seguir:

  1. Seleccione la palabra a corregir (estará marcada con rojo o azul). Vea el siguiente ejemplo:

  2. Presione el botón derecho del ratón sobre la palabra y aparecerá un menú con las posibles opciones para corregir la palabra.

  3. Seleccione la palabra que corrija el error ortográfico, para ello haga clic con el botón izquierdo del ratón sobre la palabra seleccionada.
  1. Seleccione del menú "Revisar" (Review) la opción "Ortografía y gramática" (Spelling and grammar) o bien presione la tecla "F7".

  2. Al hacer lo anterior se abrirá una ventana con los comandos disponibles:

    • Omitir una vez (ignore): Esta acción dejará la palabra como se encuentra sin hacer alguna modificación, pasando a la siguiente palabra marcada con error ortográfico o gramatical.

    • Omitir todas (ignore all): Lo que hace este comando es omitir esta palabra en todo el texto, es decir, no verificará la palabra en el resto del texto.
    • Cambiar (change): Cambia la palabra con error por la palabra que se seleccionó del listado de sugerencias.
    • Cambiar todas (change all): Cambia la palabra con error por la palabra que se seleccionó del listado anterior, pero si se encuentra la palabra con error nuevamente en el texto la cambia por la selección que se hizo.
