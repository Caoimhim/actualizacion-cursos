Obtención de Datos de la Computadora

El tercer paso en el proceso de resolución de problemas consiste en reunir datos de la computadora, como:

  1. Mensaje de error.
  2. Secuencia de sonidos.
  3. LED.
  4. POST.

Cuando se producen errores de sistema, usuario o software en una computadora, se actualiza el Visor de Eventos con la información sobre los errores.

La información que presenta es:

  1. El problema que se produjo.
  2. La fecha y la hora del problema.
  3. La gravedad del problema.
  4. El origen del problema.
  5. Número de ID del evento.
  6. El usuario que estaba conectado cuando se produjo el problema.

El administrador de dispositivos muestra todos los dispositivos configurados en una computadora.

Para reducir los problemas en el software:

    Cada fabricante de BIOS tiene una secuencia exclusiva de sonido para las fallas de hardware.

    Cuando intente resolver un problema, encienda la computadora y escuche.

    Documente la secuencia de códigos de bip y busque el código para determinar la falla específica del hardware.

Si la computadora se inicia y se detiene después de POST, deberá averiguar la configuración del BIOS para determinar dónde se encuentra el problema.

En el mercado existen muchos software disponibles que pueden ayudarnos a diagnosticar y resolver problemas.
