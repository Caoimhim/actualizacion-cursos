Aplicaciones y los Protocolos de Internet

Un protocolo es un conjunto de reglas. Los protocolos de Internet son conjuntos de reglas que rigen la comunicación dentro de las computadoras de una red y entre ellas.

Las funciones principales de los protocolos son:

  1. Identificar errores.
  2. Comprimir los datos.
  3. Decidir cómo deben enviarse los datos.
  4. Direccionar los datos.
  5. Decidir cómo anunciar los datos enviados y recibidos.

Aunque existen muchos protocolos, a continuación se resumen algunos de los protocolos más comunes que se utilizan en redes y en Internet.

Protocolos

Protocolo Descripción
TCP/IP Un protocolo utilizado para transortar datos en Internet.
NETBEUI/NETBIOS Un protocolo pequeño y rápido, diseñado para una red de grupo de trabajo que no requiere conexión a Internet.
IPX/SPX Un protocolo utilizado para transportar datos de una red Netware de Novell.
HTTP/HTTPS Un protocolo que define de qué manera se intercambian archivos en la Web.
FTP Un protocolo que brinda servicios para la transferencia y la manipulación de archivos.
SSH Un protocolo que se utiliza para conectar computadoras de manera segura.
TELNET Un protocolo que utiliza una conexión basada en texto para una computadora remota TCP/IP.
POP Un protocolo utilizado para descargar mensajes de correo electrónico de un servidor de correo electrónico.
IMAP Un protocolo utilizado para descargar mensajes de correo electrónico de un servidor de correo electrónico.
SMTP Un protocolo utilizado para enviar correo en una red TCP/IP.

Detalles de Protocolos Comunes

Protocolo Descripción
TCP/IP La suite de protocolos TCP/IP se ha convertido en el estándar dominante para networking. TCP/IP representa un conjunto de estándares públicos que especifican cómo los paquetes de informaicón se intercambian entre computadoras a través de una o más redes.
IPX/SPX Intercambio de paquetes de internetnetwork e intercambio de paquetes secuenciados es la suite de protocolos utilizada originalmente por el sistema operativo de red Novell Corporations, NetWare. Brinda funciones similares a las que se incluyen en TCP/ÍP. Novel, en sus versiones actuales, es compatible con el conjunto de aplicaciones TCP/IP. Aún existe una gran base de redes NetWare que sigue utilizando IPX/SPX.
NETBEUI/NETBIOS La interfaz de usuario NetBIOS extendida es un protocolo utilizado principalmente en redes peuqeñas de Windows NT. NetBEUI no se puede enrutar ni utilizar con routers ara la comunicación en una red grande. NetBEUI es ideal para redes peer-to-peer pequeñas que incluyen pocas computadoras directamnete conectadas entre sí. Puede utilizarse junto con otro protocolo enrutable, como TCP/IP. Esto proporciona al administrador de red las ventajas del alto rendimiento de NetBEUI dentro de la red local y la capacidad de comunciarse más allá de la LAN a través de TCP/IP.
AppleTalk Suite de protocolos para colocar en red computadores Macintosh. Está compuesta por un completo conjunto de protocolos que abarcan las siete capas del modelo de referencia OSI. El protocolo AppleTalk fue diseñado para ser ejecutado en LocalTalk, la topología física de la LAN de Apple, y en lso principales tipos de LAN, sobre todo Ethernet y Token Ring.
HTTP El protocolo de trnasferencia de hipertexto rige la manera en que se intercambian los archivos, como texto, gráficos, sonidos y video, en la Word Wide Web (WWW). El grupo de trabajo de ingeniería de Internet (IETF, Internet Engineering Task Force) desarrolló los estándares para HTTP.
FTP El protocolo de transferencia de archivos es un protocolo que ofrece servicios para transferencia y la manipulación de archivos. El FTP permite conexiones múltiples y simúltaneas con sistemas de archivos remotos.
SSH El host Secure Socket se utiliza ara coenctarse de manera segura a una computadora remota.
TELNET Aplicación que se utiliza para conectarse a una computadora remota, pero carece de funciones de seguridad.
POP El protocolo de oficina de correos se utiliza para descargar correo electrónico de un servidor de correo remoto.
IMAP El protocolo de acceso a mensajes de Internet también se utiliza para descargar correo electrónico de un servidor de correo remoto.
SMTP El protocolo simple de transferencia de correo se utiliza para enviar un correo electrónico a un servidor de correo remoto.

Cuanto más comprenda sobre cada uno de estos protocolos, más entenderá sobre el funcionamiento de las redes e Internet.
