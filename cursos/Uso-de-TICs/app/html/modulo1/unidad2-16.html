Tipos de Ventana del Explorador Windows

El Explorador de Windows consta de los siguientes tipos de ventanas:

  • Vistas en Miniatura (Thumbnails)
  • Mosaicos (Tiles)
  • Iconos (Icons)
  • Lista (List)
  • Detalles (Details)

Aparecerá una pequeña representación del contenido de los archivos con formato de imagen,
como puede ser .jpg, .jpeg, .bmp, .gif, etc.

Aquellos que tengan otro formato o sean carpetas aparecerá el icono correspondiente al tipo de archivo en dimensiones mayores.

¿Cuándo conviene utilizarla?
Este tipo de vista es útil cuando tratamos con pocas imágenes, ya que permite identificar con más facilidad el contenido de cada archivo, y por lo tanto, facilitar la búsqueda de alguna imagen en concreto.

En estas vistas se puede cambiar la posición de los elementos en la carpeta, sólo hay que arrastrarlos a la posición deseada, y los elementos se mantienen la posición que se indique, no se reorganizan los elementos aunque se agregue o elimine alguno.

Las carpetas y archivos aparecen representados con una imagen grande, a un lado se ve el nombre, el tipo de archivo y el tamaño en KB; si es una imagen aparece el tamaño en pixeles.

Los elementos se van organizando uno al lado del otro, de izquierda a derecha.

¿Cuándo conviene utilizarla?
Este tipo de iconos se utiliza cuando tenemos pocos elementos en alguna unidad o carpeta.
Con esta vista se aprecia mejor el nombre de los elementos y permite seleccionarlos con más facilidad.

En estas vistas se puede cambiar la posición de los elementos en la carpeta, sólo hay que arrastrarlos a la posición deseada, y los elementos se mantienen la posición que se indique, no se reorganizan los elementos aunque se agregue o elimine alguno.

Los archivos aparecen representados por un icono en la opción "mosaicos". El único dato que aparece debajo del icono es el nombre del archivo o carpeta.

¿Cuándo conviene utilizarla?
Estos tipos de iconos se utilizan cuando la carpeta que se tiene seleccionada contiene una cantidad media de elementos.

Aparecen iconos pequeños, uno debajo del otro, facilitando así las búsquedas por nombre, en esta vista no aparece más que el nombre de la carpeta o archivo.

¿Cuándo conviene utilizarla?
Cuando tenemos muchos archivos, y ya sabemos el nombre del archivo y el tipo de archivo. De esta forma es más fácil su búsqueda.

En estas vistas, los elementos aparecerán unos debajo de otros; en caso de eliminar o agregar algún archivo, se reorganizarán las posiciones.

Aparecen los iconos uno debajo del otro acompañado de algunas de sus propiedades. Con este tipo de vista podemos ordenar (de mayor a menor o viceversa) los elementos por: tamaño, fecha de modificación, nombre, etc.

¿Cuándo conviene utilizarla?
Este tipo de vista se utiliza cuando se quiere encontrar algún elemento que tenga determinadas características, como: tamaño, tipo de elemento, fecha de modificación, etc.

En estas vistas, los elementos aparecerán unos debajo de otros; en caso de eliminar o agregar algún archivo, se reorganizarán las posiciones.
