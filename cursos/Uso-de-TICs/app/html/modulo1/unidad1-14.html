Aplicaciones

De acuerdo a la tarea que vaya a realizar (crear documentos, realizar cálculos, elaborar presentaciones, entre otros) es el software o programa que debe utilizar.

El software se puede agrupar en tres grandes categorías:

  • Software de Sistema
  • Software de Desarrollo
  • Software de Aplicación

Conjunto de programas cuyo objeto es facilitar el uso de la computadora, ya que indica cómo utilizar sus componentes para poder llevar a cabo las instrucciones del usuario, es decir, sirve de enlace entre la computadora y el usuario.

Este tipo de software es indispensable para controlar las funciones básicas de la computadora, cuando ésta se enciende el sistema operativo se carga en la memoria y permanece ahí hasta que la computadora se apaga. Por ejemplo: el sistema operativo que tiene la computadora que está utilizando se llama Windows porque está formado de ventanas fáciles de manejar.

Algunos ejemplos de sistemas operativos son:

  • Microsoft: Windows 7, Windows 10.
  • Linux: Ubuntu.
  • Apple: Tiger, Snow Leopard.

Este tipo de software se utiliza para desarrollar los programas que se utilizan en la computadora.

En un supermercado, cada vez que un cajero nos cobra un producto, lo pasa por un lector de código que alimenta un programa que cobra productos, ese programa está realizado en un software de desarrollo bajo un cierto lenguaje de programación,

Como ejemplos de estos lenguajes tenemos: C++, Java, Delphi, Progress, Visual Basic, QBasic, entre otros.

Es un programa desarrollado bajo ciertas características que permiten la realización de una actividad o tarea determinada.

Estos programas son los más solicitados por la mayoría de usuarios de computadoras debido a los beneficios o servicios que reciven de ellos.

Estos programas son empleados por el usuario para simplificar su vida, ya sea para:

  • Escribir un libro, una tesis o un memorándum.
  • Manejar grandes volúmenes de datos.
  • Emitir una nómina de una empresa.

El software de aplicación más difundido en el mercado se puede dividir en varias categorías, las principales son:

  • Procesadores de Palabras: sirve para crear documentos.
  • Hojas de Cálculo: sirve para realizar cálculos y sus gráficas.
  • Bases de Datos: sirve para almacenar y organizar datos.
  • Presentadores: sirve para realizar presentaciones gráficas.
  • Comunicación: sirve para tener comunicación electrónica y navegar en Internet.
