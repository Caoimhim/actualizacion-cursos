El Sistema Operativo Windows

Para que una computadora funcione requiere de un software que se llama sistema operativo.

El sistema operativo es el software más importante de una computadora.

El sistema operativo:

  1. Controla y administra el hardware conectado al equipo.
  2. Proporciona una interfaz que permite interactuar con el equipo.
  3. Asegura que la computadora funcione correctamente.

Uno de los sistemas operativos más recientes es Windows 10.

El sistema operativo debe actualizarse periódicamente a fin de asegurar que es compatible con cualquier hardware nuevo que instale.
