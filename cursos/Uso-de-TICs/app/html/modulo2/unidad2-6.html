Ejercicio: Elementos de la Cuenta de Correo Electrónico

Objetivo: Identificar los elementos con los que se construye una cuenta de correo electrónico.
Instrucciones:

  1. Relacione la columna de la descripción y opciones.
  2. Al finalizar el ejercicio verifique las respuestas correctas haciendo clic en el botón "Revisar".

  1. Representa una dirección de correo electrónico de una persona.
  2. Es el texto que representa el nombre del proveedor de correo electrónico.
  3. Elemento que indica que el proveedor de correo electrónico es una organización comercial.
  4. Representa el nombre de usuario.
  5. Elemento que indica que el proveedor de correo electrónico se encuentra en México.
  6. Representa el nombre de un dominio.
  7. Se utiliza para separar el nombre de usuario del dominio donde éste se encuentra.
