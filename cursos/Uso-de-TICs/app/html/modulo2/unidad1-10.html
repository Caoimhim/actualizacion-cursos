Elementos de un Navegador

Una vez que abrió el navegador escriba la dirección electrónica (URL) que desea visitar.
Pase el cursor (ratón) por el siguiente interactivo de página electrónica y descubra los elementos de un navegador.

  • Barra de Direcciones
  • Controles de Pestaña
  • Barra de Herramientas

Aquí es donde siempre que usted quiera ingresar a una página debe escribir la dirección por ejemplo: http://www.conocer.gob.mx/ y al finalizar de escribir es necesario presionar la tecla Intro (Enter) o para tener acceso a la página.

Oprimiendo el pequeño triángulo del extremo izquierdo de la barra de direcciones, se obtiene una lista de los sitios recientemente visitados, para visitarlos de nuevo sólo haga clic sobre alguno de ellos.

En esta barra también contiene varios íconos o botones que realizan funciones específicas:

Botón ¿Qué hace?
Atrás (Back)
 Permite ir a la página que se acaba de visitar.
Delante (Forward)
 Permite ir hacia adelante, es decir, volver a avanzar lo que antes se retrocedió.
Actualizar (Refresh)
 Vuelve a cargar la página actual en dado caso de que no hay cargado completa o marque un error durante su carga en el navegador.
Búsqueda (Search)
 Este botón al ser presionado abre una ventana con los resultados proporcionados por Google a partir de las palabras se pusieron para búsqueda.

Nota: si al andar navegando se presenta una ventana con la leyenda:
No se puede mostrar la página (Cannot display the webpage) es porque está mal escrita la dirección o bien porque esa página no existe, aquí la sugerencia es: Verificar que la dirección de la página esté bien escrita.

Permiten trabajar con más de una página web en una sola ventana.

Cuando se tiene más de una ventana abierta, la navegación será más rápida pues se tendrá todo organizado en una pestaña. Para pasar de una pestaña a otra sólo tendrá que hacer clic sobre su título.

Es la fila de botones que proporciona una serie de funciones y comandos facilitando las actividades que se realizan en el navegador.

Botón ¿Qué hace?
Inicio (Home)
 Si se da clic sobre el botón permite visualizar la página de Inicio que se tiene predeterminada, permitiendo agregar o cambiar dicha página.
RSS
 Permite añadir el sitio visitado rápidamente a la lista de fuentes RSS. RSS es la tecnología que facilita estar al tanto de todo el contenido que se genera en una página.
Imprimir (Print)
 Permite imprimir la página Web que se está visualizando.
Página (Page)
 Contiene las opciones relativas de la página que se tiene en pantalla.
Herramientas (Tools)
 Muestra las diferentes opciones que se tienen en el navegador, como las opciones de Internet y la administración de menús.

Por lo que la siguiente vez que se necesite sólo basta con desplegar la lista y seleccionar la página indicada.
