¿Cómo acceder una página electrónica?

Imagine que su mejor amigo lo invita a una reunión, ¿qué es lo primero que usted le preguntaría?

Seguramente le diría: “¿Dónde será la reunión? Pásame la dirección".

Y esa dirección está compuesta por el nombre de una calle, el número, la colonia, la localidad o el municipio.

Así como nosotros pedimos la dirección para llegar a un lugar que es de nuestro interés lo mismo ocurre con las páginas Web, para localizarlas utilizamos una dirección o un URL (Universal Resource Locator) que en español sería un "Localizador Universal de Recursos de Internet", que da una sintaxis (formato) estándar para mostrar los distintos recursos de Internet.

Dirección de Página Web: Un URL de acceso a recursos Web se denota por el prefijo http:// seguido de la dirección de Internet completa del sitio al que se desea acceder.

Características

Ejemplo

El URL para acceder a la página electrónica del Sistema Nacional de Competencias es: http://www.conocer.gob.mx/

Coloque el cursor sobre el URL del Sistema Nacional de Competencias y haga clic, como podrá observar desde esta página electrónica y a través de una liga usted puede acceder a otra, que se encuentra en un Servidor diferente.
