Caso: la historia de Clara

El caso de Clara y sus amigas….asignando tareas y responsabilidades.

Problema

Clara, Hilda y Jimena planearon las actividades para iniciar su negocio, pero todas querían preparar las pizzas, no querían estar cobrando dinero, pues no tenían confianza para hacer bien las cuentas y tenían miedo de que se les perdiera el dinero. Como las tres querían trabajar en cocina, y esto no era posible, tenían dos opciones o aceptar el hacer otras actividades o contratar a personal de ayuda lo que podía salirles más caro.

Solución y siguientes acciones

Finalmente Clara aceptó y tomó la responsabilidad de realizar las compras para hacer las pizzas, pero, faltaba más gente para que se encargara de las demás actividades.
