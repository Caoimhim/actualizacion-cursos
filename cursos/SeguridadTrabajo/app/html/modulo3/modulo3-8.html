Protección respiratoria

Mascaras desechables : Estas máscaras se conocen como N95 y quiere decir que, paran el 95 % de las partículas en 0.3 micrones de diámetro.

Cuando remueva paredes con asbesto, consulte con un experto el tipo de mascarilla que debe utilizar. También utilice un respirador reusable de 1 cartucho cuando esté manejando aserrín de madera. El respirador con 2 cartuchos es para sustancias peligrosas.

Protección de manos

El equipo más común para proteger las manos son los guantes

Existen diferentes tipos de material para guantes:

  • Algodón con puntos resaltados
  • Poliester con capa reforzada.
  • Lona con piel en palma y punta de los dedos
  • Diseños anatomico de material sintetico
  • Guante de piel para manejar

Usa guantes cuando manejes herramientas pesadas

Tipos de guantes

1. Guantes especializados para químicos. Estos sirven para proteger las manos cuando se manejas químicos o solventes.


2. Guantes para trabajo mecánico. Estos dos tipos de guantes sirven para reparaciones mecánicas, soldadura y trabajo con cuchillas.
