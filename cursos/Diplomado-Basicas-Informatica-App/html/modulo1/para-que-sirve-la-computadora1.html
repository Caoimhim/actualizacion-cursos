3.2 Categorías del software

El software se puede agrupar en tres grandes categorías.

  • Sistemas
  • Desarrollo
  • Aplicación

Un software de sistema es un conjunto de programas cuyo objeto es facilitar el uso de la computadora, ya que indica cómo utilizar sus componentes para poder llevar a cabo las instrucciones del usuario, es decir, sirve como un enlace entre la computadora y el usuario.

El sistema operativo que tiene la computadora que está utilizando se llama Windows (Ventanas) porque está formado de puras ventanas fáciles de manejar. El software de sistema es indispensable para controlar las funciones básicas de la computadora, cuando ésta se enciende el sistema operativo se carga en la memoria y permanece ahí hasta que la computadora se apaga.

Algunos ejemplos son Microsoft Windows XP, Vista, 7 y el sistema operativo de la Macintosh: Mac OS, entre otros.

Un software de desarrollo es untipo de software que se utiliza para desarrollar los programas que se utilizan en la computadora.

Por ejemplo: en un supermercado cada vez que un cajero nos cobra un producto, lo pasa por un lector de código que alimenta un programa que cobra productos, ese programa está realizado en un software de desarrollo bajo un cierto lenguaje de programación.

Como ejemplos de esos lenguajes tenemos: C++, Java, .NET, PHP, Perl, ABAD (SAP) entre otros.

Un software de aplicación es un programa desarrollado bajo ciertas características que permiten la realización de una actividad o tarea determinada.

Estos programas son empleados por el usuario para simplificar su vida, ya sea si decide escribir un libro, una tesis o un memorándum, o para poder manejar grandes volúmenes de datos; o aquellos programas que le sirvan para emitir una nómina de una empresa. Estos programas son los más solicitados por la mayoría de usuarios de computadoras debido a los beneficios o servicios que reciben de ellos.

El software de aplicación más difundido en el mercado se puede dividir en varias categorías, las principales son:

  • Procesadores de palabras -> sirve para crear documentos
  • Hojas electrónicas de cálculo -> sirve para realizar cálculos
  • Bases de datos -> sirve para almacenar y organizar datos
  • Presentadores -> sirve para realizar presentaciones gráficas
  • Comunicación -> sirve para tener comunicación electrónica y navegar en Internet
