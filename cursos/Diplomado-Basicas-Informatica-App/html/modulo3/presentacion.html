Módulo 3: ¿Cómo me puede ayudar la computadora en mis actividades?

¡Bienvenido al módulo 3!

En el módulo 2 vimos que la elaboración de un documento ya impreso o bien electrónico requiere de una elaboración con diseño y formato para que sea entendible para cualquier persona que lo lea, es por ello que es importante conocer lo principal de cada herramienta para lograrlo. Dentro del módulo 1, se definieron las aplicaciones que veremos en este módulo de manera básica y a continuación se irán conociendo poco a poco más de ellas.

Microsoft Office en sus versiones 2007 y 2010 será el paquete que se enseñara y conocerá dentro de este módulo. Este paquete (varias aplicaciones) es el más usado y comprado para casa, escuela u oficina.

  • El procesador de textos se llama: Microsoft Word.
  • El presentador electrónico se llama: Microsoft Power Point.
  • Y por último la Hoja de cálculo se llama: Microsoft Excel.

Al final del modulo serás capaz de:

  • Conocer qué es el paquete Office.
  • Abrir y cerrar una aplicación de Office.
  • Identificar el área de trabajo de las aplicaciones de Office.
  • Diferenciar el menú y las herramientas de las aplicaciones de Office.
  • Experimentar con las herramientas para el manejo de documentos, medición y formato de las aplicaciones de Office.
  • Crear un documento de Word, darle formato, revisar la ortografía, insertar tablas y manejar plantillas predeterminadas.
  • Crear una presentación de Power Point, insertar diapositivas y objetos, modificar objetos, animar la presentación.
  • Crear una hoja de cálculo de Excel, modificar el libro, trabajar con los datos, diseñar gráficos y transferir datos de una hoja a otra.
