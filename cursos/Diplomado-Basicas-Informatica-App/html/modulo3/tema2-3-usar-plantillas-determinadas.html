2. Trabajando con el procesador de textos

Usar plantillas determinadas

Las plantillas son formatos disponibles en Word para realizar diferentes tipos de documentos o actividades. En este caso, veremos cómo nos puede ayudar una plantilla de Word para realizar un curriculum vitae.

¿Y que es un curriculum vitae?

El curriculum vitae se ha convertido en un documento imprescindible al momento de solicitar un trabajo, una beca o algún premio. El currículum o CV como abreviación, tiene una forma establecida que resulta muy útil conocer, ya que sirve para describir la trayectoria académica y profesional de una persona.

Microsoft Word es la herramienta en la que podemos realizarlo, puede ser con un formato personal o bien con una de las plantillas preestablecidas que tiene Word.
