Módulo 2: ¿Cómo funciona la computadora?

¡Bienvenido al módulo 2!

En este módulo vamos a aprender cómo funciona este sistema operativo Windows, que es el más utilizado en todo el mundo, en computadoras que se usan en el hogar, en escuela, negocio y otros más.

Este sistema operativo es un producto de la compañía Microsoft y tiene varias versiones en uso: Windows 7, Windows 8 y Windows 10 la diferencia es el año en que se hicieron y las funcionalidades que presentan, siendo que el Windows 10 es la versión más nueva. En este módulo aprenderemos a utilizar las tres versiones utilizadas, y le será de mucho beneficio conocer cuál es la que está utilizando para que ponga más atención en estos temas.
