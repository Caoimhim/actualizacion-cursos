3.3.2 Textos en diferentes estilos

Para insertar textos en diferentes estilos se tiene que hacer lo siguiente

1. Seleccione de las herramientas de acceso rápido, el icono , que es un Cuadro de Texto (Text box).

Office 2010

2. Una vez seleccionada esa opción, el cursor toma otra forma para insertar el texto; mueva el Mouse hacia el lado derecho de la pantalla y coloque el cursor para escribir el texto indicado.
