Funciones específicas

Hay otras funciones que podemos tener en Word.

  • Espacio entre líneas y/o párrafos
  • Buscar y reemplazar palabras
  • Salto de página

¿Usted sabe qué es el espacio entre líneas o interlineado?

Se refiere al espacio que existe entre una línea y otra. Microsoft Word nos permite modificar dos tipos de espaciado:

  • Entre líneas de un párrafo (Line spacing)
  • Entre párrafos (Paragraph spacing)

Este es el más utilizado, ya que le indicamos cuánto espacio deje entre línea y línea.

Se refiere al espacio que se deja entre párrafo y párrafo. Microsoft Word nos permite especificar un espacio antes del comienzo del párrafo y otro para terminarlo.
