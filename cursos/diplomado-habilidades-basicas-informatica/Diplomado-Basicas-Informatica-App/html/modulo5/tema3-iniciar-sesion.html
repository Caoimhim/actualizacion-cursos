5.3 Crear una cuenta e iniciar sesión

El registro es la primera etapa para crear una cuenta de correo. Una vez que ya se cuenta con una, la acción para entrar a la página de Hotmail o Gmail, se conoce como Iniciar sesión. Para hacer esto debes realizar lo siguiente:

  • Outlook
  • Gmail

Iniciar sesión

Para iniciar tu sesión lo único que debes hacer es llenar tus datos en https://outlook.live.com/, ingresando tu usuario y contraseña que estableciste cuando creaste tu cuenta.

Iniciar sesión

Para iniciar tu sesión lo único que debes hacer es llenar tus datos en www.gmail.com, ingresando tu usuario y contraseña que estableciste cuando creaste tu cuenta.
